# Install LangChain - 总结

## 核心概念

Install 是 LangChain 的**安装指南**——通过 pip 或 uv 安装核心包和提供商集成包。

## 安装方式

| 方式 | 命令 | 说明 |
|------|------|------|
| **pip** | `pip install -U langchain` | Python 标准包管理器 |
| **uv** | `uv add langchain` | 更快的现代包管理器 |

要求：Python 3.10+

## 核心包 + 集成包

LangChain 采用**核心 + 集成**的包结构：

```
langchain                  ← 核心包
├── langchain-openai       ← OpenAI 集成
├── langchain-anthropic    ← Anthropic 集成
├── langchain-google-genai ← Google Gemini 集成
└── ...                    ← 700+ 集成
```

```bash
# 核心包
pip install -U langchain

# 集成包（按需安装）
pip install -U langchain-openai
pip install -U langchain-anthropic
```

## LangSmith 追踪

```bash
export LANGSMITH_TRACING="true"
export LANGSMITH_API_KEY="..."
```

用于调试和监控 LangChain 应用。

## 关键注意事项

1. **Python 3.10+** — 最低版本要求
2. **独立集成包** — 700+ 集成各自独立安装
3. **LangSmith 可选** — 但强烈推荐用于调试
4. **`-U` 标志** — 确保安装最新版本
