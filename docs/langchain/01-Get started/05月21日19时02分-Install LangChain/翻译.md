# Install LangChain

安装 LangChain 包：

```bash
# pip
pip install -U langchain
# 需要 Python 3.10+

# uv
uv add langchain
# 需要 Python 3.10+
```

LangChain 提供了对数百个 LLM 和数千个其他集成的支持。这些集成位于独立的提供商包中。

```bash
# pip
# 安装 OpenAI 集成
pip install -U langchain-openai

# 安装 Anthropic 集成
pip install -U langchain-anthropic

# uv
# 安装 OpenAI 集成
uv add langchain-openai

# 安装 Anthropic 集成
uv add langchain-anthropic
```

> **提示：** 有关可用集成的完整列表，请参阅 [Integrations tab](/oss/python/integrations/providers/overview)。

现在你已经安装了 LangChain，可以通过[快速入门指南](/oss/python/langchain/quickstart)开始使用。

> **提示：** 设置 [LangSmith](https://smith.langchain.com) 追踪来调试你的第一个 LangChain 应用。按照[追踪快速入门](/langsmith/trace-with-langchain)开始。
