# Quickstart - 总结

## 核心概念

Quickstart 是 LangChain 的**快速入门指南**——在几分钟内构建第一个 agent，涵盖安装、配置、工具创建、记忆管理和 LangSmith 追踪。

## 快速开始流程

```
1. 安装依赖        → pip install langchain deepagents
2. 设置 API 密钥   → export OPENAI_API_KEY="..."
3. 构建基础 agent  → create_agent(model, tools, system_prompt)
4. 运行            → agent.invoke({"messages": [...]})
```

## 两种 Agent 框架

| 框架 | 用途 | 特点 |
|------|------|------|
| **LangChain agent** | 细粒度控制 | 完全自定义工具、记忆、流程 |
| **Deep agent** | 最大功能 | 内置规划、文件系统、子 agent |

```python
# LangChain agent
from langchain.agents import create_agent
agent = create_agent(model=model, tools=[...], system_prompt="...")

# Deep agent
from deepagents import create_deep_agent
deep_agent = create_deep_agent(model=model, tools=[...], system_prompt="...")
```

## 六个核心步骤

### 1. 定义系统提示
```python
SYSTEM_PROMPT = """You are a literary data assistant.
- `fetch_text_from_url`: loads document text from a URL."""
```

### 2. 创建工具
```python
from langchain.tools import tool

@tool
def fetch_text_from_url(url: str) -> str:
    """Fetch the document from a URL."""
    # 实现...
```

### 3. 配置模型
```python
from langchain.chat_models import init_chat_model
model = init_chat_model("openai:gpt-5.4", temperature=0.5, timeout=300)
```

### 4. 添加记忆
```python
from langgraph.checkpoint.memory import InMemorySaver
checkpointer = InMemorySaver()
```

### 5. 创建并运行 agent
```python
agent = create_agent(model=model, tools=[...], system_prompt="...", checkpointer=checkpointer)
result = agent.invoke({"messages": [...]}, config={"configurable": {"thread_id": "1"}})
```

### 6. 查看结果
```python
print(result["messages"][-1].content_blocks)
```

## Deep agent 的额外能力

| 能力 | 说明 |
|------|------|
| **规划** | `write_todos` 工具分解任务 |
| **文件系统** | `grep`、`read_file` 等工具管理上下文 |
| **子 agent** | 委派复杂子任务 |

## 支持的提供商

OpenAI / Google Gemini / Anthropic / OpenRouter / Fireworks / Baseten / Ollama / Azure / AWS Bedrock / HuggingFace

## LangSmith 追踪

```bash
export LANGSMITH_TRACING="true"
export LANGSMITH_API_KEY="..."
```

## 关键注意事项

1. **两种框架** — LangChain agent（控制）vs Deep agent（功能）
2. **工具即函数** — 用 `@tool` 装饰器定义
3. **记忆可选** — 生产环境用持久化检查点
4. **LangSmith 推荐** — 调试和监控 agent 调用
5. **多提供商支持** — 更换模型只需改 `model` 参数
