# DeepAgent Quickstart - 总结

## 核心定位

DeepAgent 是 LangChain 推出的高级代理框架，内置规划、文件系统、子代理等能力，开箱即用。

## 最小代码

```python
from deepagents import create_deep_agent

agent = create_deep_agent(
    model="provider:model",     # 模型
    tools=[your_tool],          # 自定义工具
    system_prompt="instructions" # 系统提示
)

result = agent.invoke({"messages": [{"role": "user", "content": "..."}]})
```

## 依赖

```bash
pip install deepagents tavily-python
```

## 内置能力

| 能力 | 说明 |
|------|------|
| 自动规划 | 使用 `write_todos` 工具分解任务 |
| 文件系统 | `write_file`、`read_file` 卸载大结果 |
| 子代理 | 自动委派复杂子任务 |
| 流式传输 | 内置实时更新 |
| 可观测性 | LangSmith 追踪 |

前提：模型必须支持 tool calling。

## 执行流程

```
用户输入
  ↓
write_todos（规划）→ 分解任务
  ↓
internet_search（搜索）→ 收集信息
  ↓
write_file/read_file（文件系统）→ 管理上下文
  ↓
子代理委派（可选）
  ↓
综合报告 → 输出
```
