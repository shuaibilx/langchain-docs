# Memory and Skills - 总结

## 核心概念

Memory and Skills 是 Deep Agents Code 的**持久化自定义系统**——通过 `AGENTS.md` 文件和自动记忆实现跨会话的知识持久化，通过 Skills 实现可复用的专业化工作流。

## 两种自定义方式

| 方式 | 用途 | 存储位置 |
|------|------|---------|
| **Memory** | 编码风格、偏好、学习到的约定 | `~/.deepagents/<agent>/memories/` |
| **Skills** | 特定任务的专业化工作流和领域知识 | `skills/` 目录下的 `SKILL.md` |

## Memory 系统

### 自动记忆协议

```
1. 研究 → 开始任务前搜索记忆
2. 响应 → 不确定时检查记忆
3. 学习 → 自动保存新信息
```

### 记忆存储

```
~/.deepagents/<agent>/memories/
├── api-conventions.md
├── database-schema.md
└── deployment-process.md
```

### AGENTS.md 文件

| 类型 | 路径 | 加载时机 |
|------|------|---------|
| **全局** | `~/.deepagents/<agent>/AGENTS.md` | 每个会话 |
| **项目** | `.deepagents/AGENTS.md` | 在该项目内运行时 |

### 全局 vs 项目 AGENTS.md

| 全局 AGENTS.md | 项目 AGENTS.md |
|---------------|---------------|
| 个性、风格 | 项目架构 |
| 通用编码偏好 | 项目特定约定 |
| 通用工具使用模式 | 测试/部署策略 |
| 通用工作流 | 团队指南 |

### 额外记忆文件

在 `.deepagents/` 中添加文件并在 `AGENTS.md` 中引用：

```markdown
# .deepagents/AGENTS.md
See [api-conventions.md](./api-conventions.md) for API standards.
```

## Skills 系统

### 技能结构

```
skills/
└── test-skill/
    ├── SKILL.md          # 必需：名称+描述+指令
    └── scripts/          # 可选：额外资源
```

### 技能发现路径（优先级从低到高）

```
1. ~/.deepagents/<agent>/skills/
2. ~/.agents/skills/
3. .deepagents/skills/
4. .agents/skills/
5. ~/.claude/skills/     （实验性）
6. .claude/skills/       （实验性）
```

### 创建技能

```bash
dcode skills create test-skill           # 用户技能
dcode skills create test-skill --project # 项目技能
```

### 安装社区技能

```bash
npx skills add vercel-labs/agent-skills --skill web-design-guidelines -a deepagents -g -y
```

### 调用技能

```bash
# 交互模式
> /skill:code-review
> /skill:code-review arg1 arg2

# 命令行
dcode --skill code-review
dcode --skill code-review -m 'review auth module'

# 管道
cat diff.txt | dcode --skill code-review
git diff | dcode --skill code-review -m 'focus on security'

# 非交互
dcode --skill code-review -n 'review this patch' -q
```

### 管理技能

```bash
dcode skills list                  # 列出用户技能
dcode skills list --project        # 列出项目技能
dcode skills info test-skill       # 技能详情
dcode skills delete test-skill     # 删除技能
```

## /remember 命令

```bash
> /remember                    # 自动回顾并更新
> /remember "always use ruff"  # 带额外上下文
```

## 关键注意事项

1. **AGENTS.md 引用** — 额外文件必须在 AGENTS.md 中引用才会被 agent 知道
2. **项目 .git** — 项目技能需要 `.git` 文件夹
3. **全局安装限制** — `-g` 仅针对默认 `agent` 目录
4. **技能发现时机** — 启动时和 `/reload` 时
5. **`--skill` + `-n`** — 非交互模式需要 `-n`
6. **记忆优先协议** — 研究→响应→学习，自动运行
