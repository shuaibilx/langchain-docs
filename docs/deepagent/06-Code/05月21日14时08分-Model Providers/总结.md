# Model Providers - 总结

## 核心概念

Model Providers 是 Deep Agents Code 的**模型提供商集成系统**——支持任何 LangChain 兼容的聊天模型提供商，通过安装提供商包 + 设置凭证即可使用。

## 快速开始

```bash
# 安装（默认包含 OpenAI、Anthropic、Gemini）
uv tool install 'deepagents-code[baseten,groq]'

# 添加额外提供商
uv tool install deepagents-code --with langchain-ollama

# 所有提供商
uv tool install 'deepagents-code[anthropic,baseten,bedrock,cohere,deepseek,fireworks,google-genai,groq,huggingface,ibm,litellm,mistralai,nvidia,ollama,openai,openrouter,perplexity,vertexai,xai]'
```

## 模型解析顺序

```
1. --model 标志          ← 最高优先级
2. [models].default      ← config.toml 长期偏好
3. [models].recent       ← 最近使用的模型（自动写入）
4. 环境自动检测           ← OPENAI_API_KEY → ANTHROPIC_API_KEY → GOOGLE_API_KEY → GOOGLE_CLOUD_PROJECT
```

## 切换模型的三种方式

| 方式     | 命令                             | 说明            |
| ------ | ------------------------------ | ------------- |
| 交互式切换器 | `/model`                       | 显示已安装提供商的模型列表 |
| 直接指定   | `/model gpt-5.5`               | 使用任何模型，不限于列表  |
| 启动时指定  | `dcode --model openai:gpt-5.5` | 命令行参数         |

## 设置默认模型

```bash
# 通过命令
dcode --default-model anthropic:claude-opus-4-7

# 查看默认
dcode --default-model

# 清除默认
dcode --clear-default-model
```

## 模型参数（三个位置）

| 位置                          | 语法       | 生命周期 |
| --------------------------- | -------- | ---- |
| 启动时 `--model-params`        | JSON 字符串 | 仅限会话 |
| 会话中 `/model --model-params` | JSON 字符串 | 仅限会话 |
| `config.toml`               | TOML     | 持久化  |

```bash
# 启动时
dcode --model anthropic:claude-opus-4-7 --model-params '{"thinking": {"type": "enabled", "budget_tokens": 10000}}'

# 会话中
/model --model-params '{"temperature": 0.7}' anthropic:claude-opus-4-7
```

```toml
# config.toml 持久化
[models.providers.anthropic.params]
thinking = { type = "enabled", budget_tokens = 10000 }
max_tokens = 16000

[models.providers.ollama.params."qwen3:4b"]
temperature = 0.5
```

## 模型切换器过滤条件

模型出现在 `/model` 切换器中需要同时满足：

1. 提供商包已安装
2. 配置文件启用 `tool_calling`
3. 模型接受并产生文本

## 模型路由器

| 路由器        | 包                      | 配置格式                 |
| ---------- | ---------------------- | -------------------- |
| OpenRouter | `langchain-openrouter` | `openrouter:<model>` |
| LiteLLM    | `langchain-litellm`    | `litellm:<model>`    |

## `DEEPAGENTS_CODE_` 前缀

任何凭证可以通过添加 `DEEPAGENTS_CODE_` 前缀限定到 Deep Agents Code：

```
DEEPAGENTS_CODE_OPENAI_API_KEY  →  优先于 OPENAI_API_KEY（仅在 dcode 内）
```

## 关键注意事项

1. **默认包含 OpenAI/Anthropic/Gemini** — 其他需要额外安装
2. **凭证存储** — `~/.deepagents/.env` 或 shell 环境变量
3. **模型解析** — `--model` > `default` > `recent` > 环境自动检测
4. **启动回退只检查 4 个凭证** — OpenAI、Anthropic、Google、Vertex AI
5. **配置文件 vs 模型列表** — 两种方式让模型出现在切换器中
6. **params 浅合并** — 每模型子表覆盖提供商级别键
