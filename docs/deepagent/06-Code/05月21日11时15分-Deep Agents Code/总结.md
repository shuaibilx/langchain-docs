# Deep Agents Code - 总结

## 核心概念

Deep Agents Code (`dcode`) 是基于 Deep Agents SDK 构建的**终端编码 agent CLI 工具**——开源、支持任意 LLM、持久记忆、可自定义技能、审批控制执行代码。

## 安装和启动

```bash
curl -LsSf https://langch.in/dcode | bash  # 脚本安装
uv tool install 'deepagents-code'           # uv 安装
dcode                                        # 启动
```

## 13 项内置能力

| 能力 | 说明 |
|------|------|
| 文件操作 | read/write/edit |
| Shell 执行 | 本地或远程 |
| 远程 sandbox | LangSmith/Daytona/Modal/Runloop/AgentCore |
| Web 搜索 | Tavily（需 API key） |
| HTTP 请求 | API 调用 |
| 任务规划 | 分解+跟踪 |
| Subagents | `task` 工具 + `AGENTS.md` |
| 记忆 | 跨会话持久化 |
| 上下文压缩 | 卸载旧消息到存储 |
| Human-in-the-loop | 审批控制 |
| Skills | 自定义专业知识 |
| MCP 工具 | 外部工具服务器 |
| LangSmith 追踪 | 可观测性 |

## 13 个内置工具

| 工具 | 需要审批 |

| `ls` / `read_file` / `glob` / `grep` | 否 |

## 两种运行模式

### 交互模式

```
dcode
> 自然语言输入
> /slash 命令
> ! shell 命令
> Shift+Tab 切换自动审批
```

### 非交互模式

```bash
dcode -n "task"                    # 单任务
echo "input" | dcode               # 管道
dcode -n "task" --max-turns 10     # 限制轮次
dcode -n "task" --timeout 120      # 限制时间
dcode -n "task" -q > output.txt    # 干净输出
dcode -n "task" -S "pytest,git"    # Shell 白名单
```

## LangSmith 追踪

```bash
# ~/.deepagents/.env
LANGSMITH_TRACING=true
LANGSMITH_API_KEY=lsv2_...
LANGSMITH_PROJECT=my-project

# 分离 agent 追踪
DEEPAGENTS_CODE_LANGSMITH_PROJECT=my-agent-traces
```

## 关键注意事项

1. **Windows 不支持** — 需要 WSL
2. **非交互模式 shell 默认禁用** — 需 `-S` 显式允许
3. **`-S all` 谨慎使用** — 允许任意 shell 命令无确认
4. **管道输入最大 10 MiB**
5. **`--max-turns`/`--timeout`** — 退出码 124（匹配 GNU `timeout`）
6. **Tavily 不通过 `/auth`** — 需手动设置 `TAVILY_API_KEY`
