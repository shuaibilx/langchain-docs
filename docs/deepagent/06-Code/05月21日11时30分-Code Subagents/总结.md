# Code Subagents - 总结

## 核心概念

Code Subagents 是 Deep Agents Code 的**自定义 subagent 定义系统**——通过 AGENTS.md 文件（带 YAML frontmatter）定义同步 subagent，让 dcode 可以将专业化任务委派给它们。

## 文件路径

| 级别 | 路径 |
|------|------|
| **项目级** | `.deepagents/agents/{name}/AGENTS.md` |
| **用户级** | `~/.deepagents/{agent}/agents/{name}/AGENTS.md` |

项目 subagent 覆盖同名用户 subagent。

## 文件格式

```markdown
---
name: researcher
description: Research topics on the web before writing content
model: anthropic:claude-haiku-4-5-20251001
---

You are a research assistant with access to web search.

## Your Process
1. Search for relevant information
2. Summarize findings clearly
```

## Frontmatter 字段

| 字段 | 必需 | 说明 |
|------|------|------|
| `name` | 是 | Subagent 名称 |
| `description` | 是 | 任务描述 |
| `model` | 否 | 覆盖主 agent 模型（`provider:model` 格式） |

Markdown 正文 → `system_prompt`

## 可配置 vs 继承

| 方式 | 可配置 |
|------|--------|
| **AGENTS.md** | `name`、`description`、`model`（system_prompt 为正文） |
| **SDK 直接** | `tools`、`middleware`、`interrupt_on`、`skills` 等全部 |

> AGENTS.md 方式定义的 subagent **继承主 agent 的工具**。

## 成本优化模式

```markdown
---
name: general-purpose
description: General-purpose agent for research and multi-step tasks
model: anthropic:claude-haiku-4-5-20251001
---

You are a general-purpose assistant. Complete the task efficiently and return a concise summary.
```

主 agent 用强模型（Opus），subagent 用便宜模型（Haiku）。

## 关键注意事项

1. **仅同步 subagent** — 异步 subagent 暂不可用
2. **项目覆盖用户** — 同名时项目级优先
3. **模型覆盖可选** — 省略 `model` 则继承主 agent
4. **工具继承** — AGENTS.md 方式无法配置工具
5. **覆盖 GP subagent** — `name: general-purpose` 覆盖内置
