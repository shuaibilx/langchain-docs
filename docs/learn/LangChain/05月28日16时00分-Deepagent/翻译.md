# Build a Data Analysis Agent from Scratch — 逐段翻译

> 原文：https://docs.langchain.com/oss/python/langchain/deep-agent-from-scratch

---

## Overview / 概览

This guide builds a data analysis agent from first principles using `create_agent` and deepagents middleware. Rather than starting with `create_deep_agent`, we assemble the harness one piece at a time: so you can see exactly what each component adds and swap in only what your use case needs.

本指南使用 `create_agent` 和 Deep Agents middleware 从第一性原理构建数据分析代理。不是从 `create_deep_agent` 开始，而是一块一块地组装 harness，这样你能看到每个组件具体添加了什么功能，只选择你需要的部分。

The agent we'll build: 我们将构建的代理：

1. Accepts a CSV file for analysis — 接受 CSV 文件进行分析
2. Writes and executes Python code in an isolated sandbox — 在隔离沙箱中编写和执行 Python 代码
3. Delegates visualization work to a specialized subagent — 将可视化工作委派给专门的子代理
4. Loads data analysis patterns from a skills file — 从技能文件加载数据分析模式

## Step 1: The minimal agent / 最小代理

A model, a loop. Nothing else yet.
一个模型，一个循环。仅此而已。

```python
from langchain.agents import create_agent

agent = create_agent("anthropic:claude-sonnet-4-6", tools=[])
```

This runs, but the agent has no filesystem and no way to execute code.
这能运行，但代理没有文件系统，也无法执行代码。

---

## Step 2: Add a sandbox backend / 添加沙箱后端

`LangSmithSandbox` gives the agent an isolated environment with a filesystem and an `execute` tool for running shell commands. The agent can install packages, write scripts, and run them—without touching the host.

`LangSmithSandbox` 为代理提供隔离环境，包含文件系统和 `execute` 工具用于运行 shell 命令。代理可以安装包、编写脚本并运行——不会影响宿主机。

```python
from langchain.agents import create_agent
from langsmith.sandbox import SandboxClient
from deepagents.backends.langsmith import LangSmithSandbox
from deepagents.middleware import FilesystemMiddleware

client = SandboxClient()
sandbox = client.create_sandbox(template_name="deepagents-deploy")
backend = LangSmithSandbox(sandbox=sandbox)

agent = create_agent(
    "anthropic:claude-sonnet-4-6",
    tools=[],
    middleware=[FilesystemMiddleware(backend=backend)],
)
```

`FilesystemMiddleware` adds `read_file`, `write_file`, `edit_file`, `glob`, and `grep`. Because `LangSmithSandbox` implements the sandbox protocol, it also adds `execute`: the agent can now run shell commands.

`FilesystemMiddleware` 添加了 `read_file`、`write_file`、`edit_file`、`glob` 和 `grep`。因为 `LangSmithSandbox` 实现了沙箱协议，它还会添加 `execute`：代理现在可以运行 shell 命令。

Upload a CSV and invoke: 上传 CSV 并调用：

```python
import csv, io

rows = [
    ["Date", "Product", "Units", "Revenue"],
    ["2025-08-01", "Widget A", 10, 250],
    ["2025-08-02", "Widget B", 5, 125],
    ["2025-08-03", "Widget A", 7, 175],
    ["2025-08-04", "Widget C", 3, 90],
]
buf = io.StringIO()
csv.writer(buf).writerows(rows)
backend.upload("sales.csv", buf.getvalue().encode())

result = agent.invoke({
    "messages": [{"role": "user", "content": "Analyze sales.csv. Summarize trends."}]
})
```

## Step 3: Add context management / 添加上下文管理

For longer analysis sessions the context window fills. `SummarizationMiddleware` compresses history automatically so the agent keeps working without hitting token limits.

对于较长的分析会话，上下文窗口会填满。`SummarizationMiddleware` 自动压缩历史记录，使代理可以持续工作而不触及 token 限制。

```python
from deepagents.middleware import FilesystemMiddleware, SummarizationMiddleware

model = "anthropic:claude-sonnet-4-6"

agent = create_agent(
    model=model,
    tools=[],
    middleware=[
        FilesystemMiddleware(backend=backend),
        SummarizationMiddleware(model=model, backend=backend),
    ],
)
```

## Step 4: Add skills / 添加技能

Skills give the agent on-demand domain knowledge via progressive disclosure: loaded only when the current task calls for it.

技能通过渐进式披露为代理提供按需领域知识：仅在当前任务需要时才加载。

Create a skill file: 创建技能文件：

```
skills/
  pandas-patterns/
    SKILL.md
```

```markdown
---
name: pandas-patterns
description: Common pandas and matplotlib patterns for data analysis and visualization
---

## Data loading
Use `pd.read_csv()` for CSV files. Always check `df.info()` and `df.describe()` first.

## Visualization
Use `matplotlib` for bar charts, `seaborn` for statistical plots.
Save figures with `plt.savefig("output.png", dpi=150, bbox_inches="tight")`.

## Reporting
Write a markdown summary to `report.md` alongside any generated charts.
```

```python
from deepagents.middleware import FilesystemMiddleware, SkillsMiddleware, SummarizationMiddleware

agent = create_agent(
    model=model,
    tools=[],
    middleware=[
        FilesystemMiddleware(backend=backend),
        SummarizationMiddleware(model=model, backend=backend),
        SkillsMiddleware(backend=backend, sources=["./skills/"]),
    ],
)
```

## Step 5: Add a visualization subagent / 添加可视化子代理

Some tasks benefit from isolation. A visualization subagent runs in its own context window, keeping chart generation separate from the main analysis—and enabling parallel execution.

某些任务适合隔离执行。可视化子代理在自己的上下文窗口中运行，将图表生成与主分析分开——并支持并行执行。

```python
from langchain.agents.middleware import TodoListMiddleware
from deepagents import SubAgent
from deepagents.middleware import (
    FilesystemMiddleware,
    SkillsMiddleware,
    SubAgentMiddleware,
    SummarizationMiddleware,
)

visualizer: SubAgent = {
    "name": "visualizer",
    "description": "Generates charts and visualizations from data files in the sandbox.",
    "system_prompt": "You are a data visualization specialist. Write Python scripts using matplotlib and seaborn. Save all figures as PNG files.",
    "tools": [],
}

agent = create_agent(
    model=model,
    tools=[],
    middleware=[
        FilesystemMiddleware(backend=backend),
        SummarizationMiddleware(model=model, backend=backend),
        SkillsMiddleware(backend=backend, sources=["./skills/"]),
        TodoListMiddleware(),
        SubAgentMiddleware(backend=backend, subagents=[visualizer]),
    ],
)
```

The main agent handles analysis and planning; it delegates chart generation to the `visualizer` subagent via the `task` tool.

主代理负责分析和规划；它通过 `task` 工具将图表生成委派给 `visualizer` 子代理。

## What you built / 你构建了什么

| Middleware | What it adds |
|------------|-------------|
| `FilesystemMiddleware` + `LangSmithSandbox` | 隔离文件系统 + `execute` 工具 |
| `SummarizationMiddleware` | 自动上下文压缩 |
| `SkillsMiddleware` | 按需加载的领域知识 |
| `TodoListMiddleware` + `SubAgentMiddleware` | 并行可视化子代理 |

This is the same foundation as `create_deep_agent`: assembled manually so you control exactly what's included. See **Prebuilt middleware** for the full list of composable capabilities, and the `create_agent` reference for all configuration options.

这和 `create_deep_agent` 的基础是一样的：手动组装让你完全控制包含哪些组件。参考 **Prebuilt middleware** 了解所有可组合的能力，参考 `create_agent` API 文档了解所有配置选项。
