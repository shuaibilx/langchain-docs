# Build a SQL Agent — 逐段翻译

> 原文：https://docs.langchain.com/oss/python/langchain/sql-agent

---

## Overview / 概览

In this tutorial, you will learn how to build an agent that can answer questions about a SQL database using LangChain agents.

本教程将教你如何使用 LangChain agents 构建一个能回答 SQL 数据库问题的代理。

At a high level, the agent will: 高层流程：

1. Fetch the available tables and schemas from the database — 从数据库获取可用表和 schema
2. Decide which tables are relevant to the question — 判断哪些表与问题相关
3. Fetch the schemas for the relevant tables — 获取相关表的 schema
4. Generate a query based on the question and schemas — 根据问题和 schema 生成查询
5. Double-check the query for common mistakes using an LLM — 用 LLM 复查常见错误
6. Execute the query and return the results — 执行查询返回结果
7. Correct mistakes until the query is successful — 纠正错误直到查询成功
8. Formulate a response based on the results — 根据结果构造回答

> **Warning:** Building Q&A systems of SQL databases requires executing model-generated SQL queries. There are inherent risks. Make sure your database connection permissions are always scoped as narrowly as possible.

> 构建 SQL 数据库问答系统需要执行模型生成的 SQL 查询。存在固有风险。确保数据库连接权限尽可能窄。

### Concepts / 概念

* **Tools** for reading from SQL databases — 用于读取 SQL 数据库的工具
* LangChain **agents** — 代理
* **Human-in-the-loop** processes — 人在回路流程

## 3. Add tools / 添加工具

### sql_db_list_tables — 列出所有表

```python
@tool
def sql_db_list_tables() -> str:
    """Input is an empty string, output is a comma-separated list of tables."""
    con = sqlite3.connect("Chinook.db")
    cursor = con.cursor()
    cursor.execute("SELECT name FROM sqlite_master WHERE type='table';")
    tables = [row[0] for row in cursor.fetchall() if not row[0].startswith("sqlite_")]
    return ", ".join(tables)
```

### sql_db_query — 执行 SQL 查询

```python
@tool
def sql_db_query(query: str) -> str:
    """Input is a detailed and correct SQL query, output is the result."""
    con = sqlite3.connect("Chinook.db")
    cursor = con.cursor()
    cursor.execute(query)
    return str(cursor.fetchall())
```

### sql_db_query_checker — 检查 SQL 查询

```python
@tool
def sql_db_query_checker(query: str) -> str:
    """Double check if your query is correct before executing it."""
    # 用 LLM 检查常见错误：
    # - NOT IN with NULL
    # - UNION vs UNION ALL
    # - BETWEEN 范围
    # - 数据类型不匹配
    # - 标识符引用
    response = model.invoke(f"Check this query for mistakes:\n{query}")
    return response.text.strip()
```

## 4. Use create_agent / 创建代理

```python
from langchain.agents import create_agent

system_prompt = """
You are an agent designed to interact with a SQL database.
Given an input question, create a syntactically correct {dialect} query to run,
then look at the results and return the answer. Always limit your query to at most {top_k} results.

You MUST double check your query before executing it.
DO NOT make any DML statements (INSERT, UPDATE, DELETE, DROP etc.).

To start you should ALWAYS look at the tables in the database.
Then you should query the schema of the most relevant tables.
""".format(dialect="sqlite", top_k=5)

agent = create_agent(model, tools, system_prompt=system_prompt)
```

---

## 5. Run the agent / 运行代理

```python
question = "Which genre on average has the longest tracks?"

for step in agent.stream(
    {"messages": [{"role": "user", "content": question}]},
    stream_mode="values",
):
    step["messages"][-1].pretty_print()
```

代理执行流程：

```
用户: "Which genre on average has the longest tracks?"
    │
    ├── Ai → sql_db_list_tables() → "Album, Artist, Customer, ..."
    ├── Ai → sql_db_schema("Track, Genre") → 表结构 + 示例数据
    ├── Ai → sql_db_query_checker("SELECT ...") → 检查无误
    ├── Ai → sql_db_query("SELECT ...") → [('Sci Fi & Fantasy', 2911783), ...]
    └── Ai → "Sci Fi & Fantasy has the longest average track length"
```

## 6. Human-in-the-loop / 人在回路

It can be prudent to check the agent's SQL queries before execution.
在执行前检查代理的 SQL 查询是审慎的做法。

```python
from langchain.agents.middleware import HumanInTheLoopMiddleware
from langgraph.checkpoint.memory import InMemorySaver

agent = create_agent(
    model, tools,
    system_prompt=system_prompt,
    middleware=[
        HumanInTheLoopMiddleware(
            interrupt_on={"sql_db_query": True},  # 只拦截 sql_db_query
            description_prefix="Tool execution pending approval",
        ),
    ],
    checkpointer=InMemorySaver(),
)
```

Run with thread_id: 带 thread_id 运行：

```python
config = {"configurable": {"thread_id": "1"}}

for step in agent.stream(
    {"messages": [{"role": "user", "content": question}]},
    config,
    stream_mode="values",
):
    if "__interrupt__" in step:
        print("INTERRUPTED:")
        interrupt = step["__interrupt__"][0]
        for request in interrupt.value["action_requests"]:
            print(request["description"])
            # 输出: Tool: sql_db_query, Args: {'query': 'SELECT ...'}
```

Resume with approval: 批准后继续：

```python
from langgraph.types import Command

for step in agent.stream(
    Command(resume={"decisions": [{"type": "approve"}]}),
    config,
    stream_mode="values",
):
    step["messages"][-1].pretty_print()
```
