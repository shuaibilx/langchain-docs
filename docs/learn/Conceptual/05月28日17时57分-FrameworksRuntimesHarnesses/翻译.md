# Frameworks, Runtimes, and Harnesses — 逐段翻译

> 原文：https://docs.langchain.com/oss/concepts/products

---

LangChain maintains several open source packages to help you build agents. Each serves a different purpose in the agent development stack.

LangChain 维护多个开源包来帮助构建代理。每个在代理开发栈中服务于不同目的。

## 三层架构

| 维度 | Framework (框架) | Runtime (运行时) | Harness (工具套件) |
|------|-----------------|-----------------|-------------------|
| 价值 | 抽象 + 集成 | 持久执行 + 流式 + HITL + 持久化 | 预定义工具 + 提示词 + 子代理 |
| 适用 | 快速启动、标准化团队构建 | 低级控制、长时间有状态工作流 | 更自主的代理、复杂非确定性任务 |
| 代表 | LangChain, Vercel AI SDK, CrewAI, OpenAI Agents SDK, Google ADK, LlamaIndex | LangGraph, Temporal, Inngest | Deep Agents SDK, Claude Agent SDK, Manus |

## Agent Frameworks (如 LangChain)

Agent frameworks provide abstractions that make it easier to get started when building with LLMs.

代理框架提供抽象，使 LLM 构建更容易上手。

LangChain is an agent framework that provides abstractions like structured content blocks, the agent loop, and middleware. LangChain is built on top of LangGraph, but you don't need to know LangGraph to use LangChain.

LangChain 是一个代理框架，提供结构化内容块、代理循环和中间件等抽象。LangChain 构建在 LangGraph 之上，但使用 LangChain 不需要了解 LangGraph。

**When to use:** 快速构建代理、需要标准抽象、构建简单代理应用。

## Agent Runtimes (如 LangGraph)

Agent runtimes provide the tooling for running agents in production.

代理运行时提供在生产环境中运行代理的工具。

Supported tools: 支持的工具：
* **Durable execution** — 持久执行：代理在故障后恢复
* **Streaming** — 流式传输
* **Human-in-the-loop** — 人在回路
* **Persistence** — 线程级和跨线程持久化
* **Low-level control** — 低级控制

LangGraph is a low-level orchestration framework and runtime for building, managing, and deploying long-running, stateful agents. Agent frameworks are generally higher level and run on agent runtimes. LangChain 1.0 is built on top of LangGraph.

LangGraph 是用于构建、管理和部署长时间运行的有状态代理的低级编排框架和运行时。代理框架通常更高级，运行在代理运行时之上。LangChain 1.0 构建在 LangGraph 之上。

**When to use:** 需要细粒度控制、需要持久执行、构建复杂工作流、需要生产就绪基础设施。

## Agent Harnesses (如 Deep Agents SDK)

Agent harnesses are opinionated, batteries-included frameworks with built-in tools and capabilities.

代理工具套件是带有内置工具和能力的有主见的、开箱即用的框架。

Supported tools: 支持的工具：
* **Planning capabilities** — 规划能力：待办列表
* **Task delegation** — 任务委派：子代理
* **File system** — 文件系统：可插拔存储后端
* **Token management** — Token 管理：历史摘要和大结果驱逐

The Deep Agents SDK builds on top of LangGraph and adds planning capabilities, file systems for context management, the ability to spawn subagents, and more.

Deep Agents SDK 构建在 LangGraph 之上，添加了规划能力、用于上下文管理的文件系统、生成子代理的能力等。

**When to use:** 长时间运行的代理、复杂多步骤任务、需要预定义工具和提示词。

## Feature Comparison / 功能对比

| Feature | LangChain | LangGraph | Deep Agents |
|---------|-----------|-----------|-------------|
| Short-term memory | LangChain 短期记忆 | LangGraph 短期记忆 | StateBackend |
| Long-term memory | LangChain 长期记忆 | LangGraph 长期记忆 | Deep Agents 记忆 |
| Skills | Multi-agent skills | — | Deep Agents Skills |
| Subagents | Multi-agent subagents | Subgraphs | Deep Agents Subagents |
| Human-in-the-loop | HITL middleware | Interrupts | interrupt_on 参数 |
| Streaming | Agent Streaming | LangGraph Streaming | Deep Agents Streaming |
