# Providers and Models — 总结

## 核心概念

```
你的代码
    │
    ▼ init_chat_model("provider:model")
LangChain 统一 API
    │
    ├── openai:gpt-5.4       → langchain-openai
    ├── anthropic:claude-*    → langchain-anthropic
    ├── google-genai:gemini-* → langchain-google-genai
    ├── openrouter:*          → langchain-openrouter
    └── ollama:*              → langchain-ollama
```

## 两种初始化方式

```python
# 方式 1: Provider prefix (推荐)
model = init_chat_model("openai:gpt-5.4")

# 方式 2: 直接实例化
from langchain_openai import ChatOpenAI
model = ChatOpenAI(model="gpt-5.4")
```

## 提供商包速查

| 提供商 | 包名 | 环境变量 |
|--------|------|---------|
| OpenAI | langchain-openai | OPENAI_API_KEY |
| Anthropic | langchain-anthropic | ANTHROPIC_API_KEY |
| Google | langchain-google-genai | GOOGLE_API_KEY |
| AWS Bedrock | langchain-aws | AWS credentials |
| Ollama | langchain-ollama | 无需 |
| Groq | langchain-groq | GROQ_API_KEY |

## 路由器/代理

| 路由器 | 用途 |
|--------|------|
| OpenRouter | 多提供商统一访问 |
| LiteLLM | 100+ 提供商，含路由和回退 |

## 关键优势

* **即时使用新模型** — 无需更新 LangChain
* **统一接口** — invoke, stream, bind_tools 跨提供商一致
* **自由切换** — 一行代码更换提供商
