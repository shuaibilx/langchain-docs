# Providers and Models — 逐段翻译

> 原文：https://docs.langchain.com/oss/concepts/providers-and-models

---

LangChain gives you a single, unified API to work with models from any provider. Install a provider package, pick a model name, and start building—the same code works whether you use OpenAI, Anthropic, Google, or any other supported provider.

LangChain 提供单一统一 API 来使用任何提供商的模型。安装提供商包、选择模型名称即可开始构建——相同的代码适用于 OpenAI、Anthropic、Google 或任何其他支持的提供商。

## One API for any model / 一个 API 适用所有模型

Every LangChain chat model, regardless of provider, implements the same interface.

每个 LangChain 聊天模型，无论提供商，都实现相同的接口。

* **Swap providers** without rewriting application logic — 无需重写应用逻辑即可切换提供商
* **Compare models** side-by-side with identical code — 用相同代码并排比较模型
* **Use advanced features** like tool calling, structured output, and streaming across all providers — 跨所有提供商使用高级功能

```python
from langchain.chat_models import init_chat_model

openai_model = init_chat_model("openai:gpt-5.4")
anthropic_model = init_chat_model("anthropic:claude-opus-4-6")
google_model = init_chat_model("google-genai:gemini-3.1-pro-preview")
```

## What is a provider? / 什么是提供商？

A provider is a company or platform that hosts AI models and exposes them through an API.

提供商是托管 AI 模型并通过 API 暴露它们的公司或平台。

Each provider has a dedicated integration package: 每个提供商有专用集成包：

```shell
langchain-openai       # OpenAI
langchain-anthropic    # Anthropic
langchain-google-genai # Google
```

## Find model names / 查找模型名称

Two ways to specify a model: 两种指定模型的方式：

```python
# Provider prefix format
model = init_chat_model("openai:gpt-5.4")

# Direct class instantiation
from langchain_openai import ChatOpenAI
model = ChatOpenAI(model="gpt-5.4")
```

| Provider | Where to find model names |
|----------|--------------------------|
| OpenAI | platform.openai.com/docs/models |
| Anthropic | docs.anthropic.com/en/docs/about-claude/models |
| Google | ai.google.dev/gemini-api/docs/models |
| AWS Bedrock | docs.aws.amazon.com/bedrock/latest/userguide/models-supported.html |
| Ollama | ollama.com/library |

## Use new models immediately / 立即使用新模型

Because provider packages pass model names directly to the provider's API, you can use new models the moment a provider releases them (no LangChain update required).

因为提供商包直接将模型名称传递给提供商的 API，所以提供商发布新模型时即可使用（无需 LangChain 更新）。

## Routers and proxies / 路由器和代理

Routers give you access to models from multiple providers through a single API and credential.

路由器通过单一 API 和凭证访问多个提供商的模型。

| Provider | Description |
|----------|-------------|
| OpenRouter | 统一访问 OpenAI, Anthropic, Google, Meta 等模型 |
| LiteLLM | 100+ 提供商的统一接口，含路由、回退和花费追踪 |

```python
model = init_chat_model("openrouter:anthropic/claude-sonnet-4-6")
```

## OpenAI-compatible endpoints / OpenAI 兼容端点

Many providers offer endpoints compatible with OpenAI's Chat Completions API.

许多提供商提供与 OpenAI Chat Completions API 兼容的端点。

```python
from langchain_openai import ChatOpenAI
model = ChatOpenAI(base_url="https://your-provider.com/v1", api_key="...", model="...")
```
