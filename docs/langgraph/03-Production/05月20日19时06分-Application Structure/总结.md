# Application Structure 文档总结

## 一句话概述

LangGraph 应用由图、配置文件（`langgraph.json`）、依赖文件和环境变量四部分组成，用于部署到 LangSmith Deployment 平台。

---

## 四个组成部分

| 组件 | 文件 | 作用 |
|------|------|------|
| 配置文件 | `langgraph.json` | 指定依赖、图、环境变量 |
| 图 | `agent.py` 等 | 实现应用逻辑 |
| 依赖 | `requirements.txt` / `pyproject.toml` | Python 包依赖 |
| 环境变量 | `.env` | API Key 等配置 |

---

## 典型目录结构

```
my-app/
├── my_agent/
│   ├── utils/
│   │   ├── tools.py       # 工具
│   │   ├── nodes.py       # 节点函数
│   │   └── state.py       # 状态定义
│   └── agent.py           # 图构建
├── .env
├── requirements.txt
└── langgraph.json
```

---

## langgraph.json 配置

```json
{
  "dependencies": ["langchain_openai", "./your_package"],
  "graphs": {
    "my_agent": "./your_package/your_file.py:agent"
  },
  "env": "./.env"
}
```

| 键 | 作用 |
|----|------|
| `dependencies` | Python 包和本地包路径 |
| `graphs` | 图名称 → 文件路径:变量名 |
| `env` | 环境变量文件路径 |
| `dockerfile_lines` | 额外系统依赖 |

---
