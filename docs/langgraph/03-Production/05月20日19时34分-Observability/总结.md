# Observability 文档总结

## 一句话概述

LangSmith Observability 通过追踪（Traces）可视化应用执行步骤，支持调试、评估和监控，可选择性追踪、添加元数据和屏蔽敏感数据。

---

## 核心功能

| 功能 | 用途 |
|------|------|
| 追踪 | 可视化执行步骤 |
| 调试 | 查看本地运行的应用 |
| 评估 | 评估应用性能 |
| 监控 | 仪表板监控 |

---

## 启用追踪

```bash
export LANGSMITH_TRACING=true
export LANGSMITH_API_KEY=<your-api-key>
```

---

## 四种配置方式

### 1. 选择性追踪

```python
with ls.tracing_context(enabled=True):
    agent.invoke(...)  # 被追踪
agent.invoke(...)      # 不被追踪
```

### 2. 项目名称

```bash
# 静态
export LANGSMITH_PROJECT=my-project

# 动态
with ls.tracing_context(project_name="test"): ...
```

### 3. 元数据和标签

```python
agent.invoke(input, config={
    "tags": ["production", "v1.0"],
    "metadata": {"user_id": "123"}
})
```

### 4. 匿名器

```python
anonymizer = create_anonymizer([
    {"pattern": r"\b\d{3}-?\d{2}-?\d{4}\b", "replace": "<ssn>"}
])
tracer = LangChainTracer(client=Client(anonymizer=anonymizer))
graph.with_config({'callbacks': [tracer]})
```

---

## 关键 API

```python
import langsmith as ls

# 选择性追踪
ls.tracing_context(enabled=True)

# 项目和元数据
ls.tracing_context(project_name="test", tags=["prod"], metadata={...})

# 匿名器
from langsmith.anonymizer import create_anonymizer
create_anonymizer([{"pattern": "...", "replace": "..."}])
```
