# LangSmith Deployment（部署）

本指南展示如何将代理部署到 **[LangSmith Cloud](/langsmith/deploy-to-cloud)**，这是一个专为代理工作负载设计的完全托管托管平台。通过 Cloud 部署，你可以直接从 GitHub 仓库部署——LangSmith 处理基础设施、扩展和运维问题。

传统托管平台是为无状态、短暂的 Web 应用构建的。LangSmith Cloud 是**专为有状态、长时间运行的代理**构建的，需要持久状态和后台执行。

> **提示：** LangSmith 提供多种部署选项，包括使用[控制平面（混合/自托管）](/langsmith/deploy-with-control-plane)部署或作为[独立服务器](/langsmith/deploy-standalone-server)部署。更多信息参见[部署概述](/langsmith/deployment)。

## 前提条件

开始之前，确保你有：

* [GitHub 账户](https://github.com/)
* [LangSmith 账户](https://smith.langchain.com)（免费注册）

## 部署你的代理

### 1. 在 GitHub 上创建仓库

你的应用代码必须位于 GitHub 仓库中才能部署到 LangSmith。支持公共和私有仓库。首先确保你的应用兼容 LangGraph，按照[本地服务器设置指南](/oss/python/langgraph/studio#设置本地-agent-server)操作。然后将代码推送到仓库。

### 2. 部署到 LangSmith

1. 登录 [LangSmith](https://smith.langchain.com)。在左侧边栏选择 **Deployments**。
2. 点击 **+ New Deployment** 按钮。
3. 如果是首次用户或添加之前未连接的私有仓库，点击 **Add new account** 按钮连接你的 GitHub 账户。
4. 选择你的应用仓库。点击 **Submit** 部署。这可能需要约 15 分钟完成。

### 3. 在 Studio 中测试应用

部署完成后：

1. 选择刚创建的部署查看详情。
2. 点击右上角的 **Studio** 按钮。Studio 将打开并显示你的图。

### 4. 获取部署的 API URL

1. 在 LangGraph 的 **Deployment details** 视图中，点击 **API URL** 复制到剪贴板。

### 5. 测试 API

**Python SDK：**

```bash
pip install langgraph-sdk
```

```python
from langgraph_sdk import get_sync_client

client = get_sync_client(url="your-deployment-url", api_key="your-langsmith-api-key")

for chunk in client.runs.stream(
    None,    # 无线程运行
    "agent", # 代理名称，定义在 langgraph.json 中
    input={
        "messages": [{
            "role": "human",
            "content": "What is LangGraph?",
        }],
    },
    stream_mode="updates",
):
    print(f"Receiving new event of type: {chunk.event}...")
    print(chunk.data)
    print("\n\n")
```

**REST API：**

```bash
curl -s --request POST \
    --url <DEPLOYMENT_URL>/runs/stream \
    --header 'Content-Type: application/json' \
    --header "X-Api-Key: <LANGSMITH API KEY>" \
    --data '{
        "assistant_id": "agent",
        "input": {
            "messages": [
                {
                    "role": "human",
                    "content": "What is LangGraph?"
                }
            ]
        },
        "stream_mode": "updates"
    }'
```
