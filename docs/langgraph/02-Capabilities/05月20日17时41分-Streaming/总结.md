# Streaming 文档总结

## 一句话概述

LangGraph 的 stream-mode API 通过 7 种流模式（updates、values、messages、custom、checkpoints、tasks、debug）暴露图执行事件，v2 格式提供统一的输出结构。

---

## 7 种流模式

| 模式 | 内容 | 需要检查点器 |
|------|------|-------------|
| `updates` | 每步状态增量 | 否 |
| `values` | 每步完整状态 | 否 |
| `messages` | LLM token 流 | 否 |
| `custom` | 自定义数据 | 否 |
| `checkpoints` | 检查点事件 | 是 |
| `tasks` | 任务开始/完成 | 是 |
| `debug` | 全部信息 | 是 |

---

## v2 统一输出格式

```python
{
    "type": "updates",  # 流模式类型
    "ns": (),           # 命名空间（子图时填充）
    "data": {...}       # 实际负载
}
```

使用 `version="v2"` 启用，支持类型收窄。

---

## 核心用法

```python
# 单模式
for chunk in graph.stream(input, stream_mode="updates", version="v2"):
    if chunk["type"] == "updates":
        print(chunk["data"])

# 多模式
for chunk in graph.stream(input, stream_mode=["updates", "messages"], version="v2"):
    if chunk["type"] == "updates": ...
    elif chunk["type"] == "messages": ...
```

---

## messages 模式详解

输出元组 `(message_chunk, metadata)`：

```python
msg, metadata = chunk["data"]
metadata["langgraph_node"]  # 节点名
metadata["tags"]            # 标签
```

过滤方式：
- 按标签：`metadata["tags"] == ["joke"]`
- 按节点：`metadata["langgraph_node"] == "node_name"`
- 排除流：`tags=["nostream"]`

---

## custom 模式

节点/工具中发出自定义数据：

```python
def node(state):
    writer = get_stream_writer()
    writer({"progress": 50})
    return state
```

适用于任意 LLM API（非 LangChain 集成）。

---

## 子图流式传输

```python
graph.stream(input, subgraphs=True, stream_mode="updates", version="v2")
```

`chunk["ns"]` 标识来源：`()` 根图，`("node:task_id",)` 子图。

---

## v1 → v2 迁移

| 变化 | v1 | v2 |
|------|----|----|
| 输出格式 | 因模式而异 | 统一 StreamPart |
| invoke 返回 | dict | GraphOutput |
| 中断 | `__interrupt__` 键 | `.interrupts` 属性 |
| 类型收窄 | 不支持 | 支持 |

---

## 与 Event Streaming 的关系

| 特性 | Streaming (本文) | Event Streaming |
|------|-----------------|-----------------|
| API | `stream_mode` 参数 | `stream_events()` |
| 投影 | 无，按 type 分支 | 8 种类型化投影 |
| 转换器 | 无 | StreamTransformer |
| 推荐 | 低级访问 | 新应用推荐 |

---

## 关键 API

```python
# 同步流
graph.stream(input, stream_mode="updates", version="v2")

# 异步流
graph.astream(input, stream_mode="messages", version="v2")

# invoke v2
result = graph.invoke(input, version="v2")
result.value       # 输出
result.interrupts  # 中断
```
