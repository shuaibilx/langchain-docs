# 使用 Graph API

本指南演示 LangGraph Graph API 的基础知识。涵盖状态定义和更新，以及常见图结构如序列、分支和循环的组合。还涵盖 LangGraph 的控制功能，包括用于 map-reduce 工作流的 Send API 和用于组合状态更新与节点间"跳转"的 Command API。

## 定义和更新状态

### 定义状态

LangGraph 中的状态可以是 `TypedDict`、`Pydantic` 模型或 dataclass。

```python
from langchain.messages import AnyMessage
from typing_extensions import TypedDict

class State(TypedDict):
    messages: list[AnyMessage]
    extra_field: int
```

### 更新状态

节点只是读取图状态并更新它的 Python 函数。

```python
from langchain.messages import AIMessage

def node(state: State):
    messages = state["messages"]
    new_message = AIMessage("Hello!")
    return {"messages": messages + [new_message], "extra_field": 10}
```

> **警告：** 节点应该直接返回状态更新，而不是修改状态。

### 用 Reducers 处理状态更新

```python
from typing_extensions import Annotated

def add(left, right):
    return left + right

class State(TypedDict):
    messages: Annotated[list[AnyMessage], add]
    extra_field: int
```

现在节点可以简化：

```python
def node(state: State):
    new_message = AIMessage("Hello!")
    return {"messages": [new_message], "extra_field": 10}
```

#### MessagesState

LangGraph 包含内置 reducer `add_messages` 和预构建的 `MessagesState`：

```python
from langgraph.graph import MessagesState

class State(MessagesState):
    extra_field: int
```

### 用 Overwrite 绕过 Reducer

```python
from langgraph.types import Overwrite

def replace_messages(state: State):
    return {"messages": Overwrite(["replacement message"])}
```

### 定义输入和输出 Schema

```python
class InputState(TypedDict):
    question: str

class OutputState(TypedDict):
    answer: str

class OverallState(InputState, OutputState):
    pass

builder = StateGraph(OverallState, input_schema=InputState, output_schema=OutputState)
```

### 在节点间传递私有状态

```python
class OverallState(TypedDict):
    a: str

class Node1Output(TypedDict):
    private_data: str

def node_1(state: OverallState) -> Node1Output:
    return {"private_data": "set by node_1"}

class Node2Input(TypedDict):
    private_data: str

def node_2(state: Node2Input) -> OverallState:
    return {"a": "set by node_2"}
```

### 使用 Pydantic 模型作为图状态

```python
from pydantic import BaseModel

class OverallState(BaseModel):
    a: str

builder = StateGraph(OverallState)
```

## 添加运行时配置

```python
from langgraph.runtime import Runtime

class ContextSchema(TypedDict):
    my_runtime_value: str

def node(state: State, runtime: Runtime[ContextSchema]):
    if runtime.context["my_runtime_value"] == "a":
        return {"my_state_value": 1}

builder = StateGraph(State, context_schema=ContextSchema)
graph.invoke({}, context={"my_runtime_value": "a"})
```

## 添加重试策略

```python
from langgraph.types import RetryPolicy

builder.add_node("node_name", node_function, retry_policy=RetryPolicy())
```

## 设置节点超时

```python
from langgraph.types import TimeoutPolicy

builder.add_node("model", call_model, timeout=TimeoutPolicy(run_timeout=120, idle_timeout=30))
```

## 处理节点错误

```python
from langgraph.errors import NodeError
from langgraph.types import Command

def error_handler(state: State, error: NodeError) -> Command:
    return Command(update={"status": "compensated"}, goto="finalize")

builder.add_node("node", func, error_handler=error_handler)
```

## 添加节点缓存

```python
from langgraph.types import CachePolicy
from langgraph.cache.memory import InMemoryCache

builder.add_node("node", func, cache_policy=CachePolicy(ttl=120))
graph = builder.compile(cache=InMemoryCache())
```

## 创建步骤序列

```python
builder = StateGraph(State)
builder.add_node(step_1)
builder.add_node(step_2)
builder.add_node(step_3)
builder.add_edge(START, "step_1")
builder.add_edge("step_1", "step_2")
builder.add_edge("step_2", "step_3")
```

内置简写：

```python
builder = StateGraph(State).add_sequence([step_1, step_2, step_3])
builder.add_edge(START, "step_1")
```

## 创建分支

### 并行执行

```python
builder.add_edge(START, "a")
builder.add_edge("a", "b")
builder.add_edge("a", "c")
builder.add_edge("b", "d")
builder.add_edge("c", "d")
```

### 延迟执行

```python
builder.add_node("d", d, defer=True)  # 等待所有前置任务完成
```

### 条件分支

```python
def route(state: State) -> Literal["b", "c"]:
    return state["which"]

builder.add_conditional_edges("a", route)
```

## Map-Reduce 和 Send API

```python
from langgraph.types import Send

def continue_to_jokes(state):
    return [Send("generate_joke", {"subject": s}) for s in state["subjects"]]

builder.add_conditional_edges("generate_topics", continue_to_jokes, ["generate_joke"])
```

## 创建和控制循环

```python
def route(state: State) -> Literal["b", END]:
    if termination_condition(state):
        return END
    return "b"

builder.add_conditional_edges("a", route)
builder.add_edge("b", "a")
```

### 设置递归限制

```python
from langgraph.errors import GraphRecursionError

try:
    graph.invoke(inputs, {"recursion_limit": 4})
except GraphRecursionError:
    print("Recursion Error")
```

### 返回状态而非抛出错误

```python
from langgraph.managed import RemainingSteps

class State(TypedDict):
    remaining_steps: RemainingSteps

def route(state):
    if state["remaining_steps"] <= 2:
        return END
    return "b"
```

## 用 Command 组合控制流和状态更新

```python
from langgraph.types import Command

def my_node(state: State) -> Command[Literal["my_other_node"]]:
    return Command(
        update={"foo": "bar"},
        goto="my_other_node"
    )
```

### 导航到父图中的节点

```python
def my_node(state: State) -> Command[Literal["my_other_node"]]:
    return Command(
        update={"foo": "bar"},
        goto="other_subgraph",
        graph=Command.PARENT
    )
```

### 在工具中使用

```python
@tool
def lookup_user_info(runtime: ToolRuntime):
    user_info = get_user_info(runtime.server_info.user.identity)
    return Command(
        update={
            "user_info": user_info,
            "messages": [ToolMessage("成功", tool_call_id=runtime.tool_call_id)]
        }
    )
```
