# Overview

> 将 LangGraph agent 渲染到前端

构建能够实时可视化 LangGraph pipeline 的前端。这些模式展示了如何渲染多步骤图执行，包含每个节点的状态和来自自定义 `StateGraph` 工作流的流式内容。

## 架构

LangGraph 图由通过边连接的命名节点组成。每个节点执行一个步骤（分类、研究、分析、综合）并将输出写入特定的状态键。在前端，`useStream` 提供对节点输出、流式 token 和图元数据的响应式访问，让你可以将每个节点映射到一个 UI 卡片。

```mermaid
graph LR
  FRONTEND["useStream()"]
  GRAPH["StateGraph"]
  N1["Node A"]
  N2["Node B"]
  N3["Node C"]

  GRAPH --"stream"--> FRONTEND
  FRONTEND --"submit"--> GRAPH
  GRAPH --> N1
  N1 --> N2
  N2 --> N3
```

```python
from langgraph.graph import StateGraph, MessagesState, START, END

class State(MessagesState):
    classification: str
    research: str
    analysis: str

graph = StateGraph(State)
graph.add_node("classify", classify_node)
graph.add_node("research", research_node)
graph.add_node("analyze", analyze_node)
graph.add_edge(START, "classify")
graph.add_edge("classify", "research")
graph.add_edge("research", "analyze")
graph.add_edge("analyze", END)

app = graph.compile()
```

在前端，`useStream` 暴露 `stream.values` 用于已完成的节点输出，暴露 `getMessagesMetadata` 用于识别哪个节点产生了每个流式 token。

```ts
import { useStream } from "@langchain/react";

function Pipeline() {
  const stream = useStream<typeof graph>({
    apiUrl: "http://localhost:2024",
    assistantId: "pipeline",
  });

  const classification = stream.values?.classification;
  const research = stream.values?.research;
  const analysis = stream.values?.analysis;
}
```

## 模式

- **Graph execution** — 可视化多步骤图 pipeline，包含每个节点的状态和流式内容。

## 相关模式

[LangChain 前端模式](/oss/python/langchain/frontend/overview)——markdown 消息、工具调用、乐观更新等——适用于任何 LangGraph 图。`useStream` hook 提供相同的核心 API，无论你使用 `createAgent`、`createDeepAgent` 还是自定义 `StateGraph`。
