# Overview - 总结

## 核心概念

Overview 是 LangGraph 前端集成的**概述页**——介绍如何将 LangGraph 的图执行可视化到前端 UI，核心是 `useStream` hook。

## 架构

```
前端 (useStream)                后端 (StateGraph)
┌──────────────┐              ┌──────────────────┐
│ stream.values │←──stream────│ Node A → Node B → Node C │
│ messages      │              │ classify → research → analyze │
│ metadata      │──submit────→│                      │
└──────────────┘              └──────────────────┘
```

## 核心 API

| API | 用途 |
|-----|------|
| `useStream` | 响应式访问节点输出、流式 token、图元数据 |
| `stream.values` | 已完成节点的输出 |
| `getMessagesMetadata` | 识别哪个节点产生了每个流式 token |

## 前端框架支持

| 框架 | 包 |
|------|------|
| React | `@langchain/react` |
| Vue | `@langchain/vue` |
| Svelte | `@langchain/svelte` |
| Angular | `@langchain/angular` |

## 关键设计

1. **命名节点** — 每个节点有名称，输出写入特定状态键
2. **流式传输** — token 实时到达前端
3. **元数据路由** — `langgraph_node` 字段标识 token 来源
4. **统一 API** — `useStream` 适用于所有 agent 类型

## 关键注意事项

1. **useStream 是核心** — 所有前端模式的基础
2. **stream.values** — 已完成节点输出
3. **getMessagesMetadata** — token 到节点的路由
4. **无需后端配置** — `langgraph_node` 自动设置
5. **适用于所有 agent** — createAgent / createDeepAgent / 自定义 StateGraph
